# 🚀 GitHub Issues Tracker

A modern and responsive **Issues Tracker Web Application** built using **HTML, CSS, JavaScript, Tailwind CSS, and DaisyUI**.  
This project simulates a real-world issue management system where users can explore, search, and analyze issues dynamically from an external API.

---

## 🌐 Live Demo

🔗 https://shoyon-at-git.github.io/B13-Assignment-5/

---

## 🧠 Project Overview

This application provides a simplified version of a GitHub-like issue tracking system.  
Users can log in using demo credentials and interact with dynamically loaded issue data including filtering, searching, and viewing detailed information.

---

## ✨ Key Features

- 🔐 Demo Login System with validation  
- 📡 Dynamic data fetching from API  
- 🔍 Search functionality for issues  
- 🎯 Filter issues by **All / Open / Closed**  
- 📊 Real-time issue count updates  
- 📄 Detailed issue view using modal  
- ⏳ Loading spinner while fetching data  
- 📱 Fully responsive UI design
